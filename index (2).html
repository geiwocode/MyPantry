What you're probably wondering
QHow does MyPantry work?
It stores saved recipes in one searchable place. You add recipes manually, and MyPantry indexes the content so you can find anything instantly using natural language.
QDo I need to manually add recipes?
Yes — V1 uses manual input while validating the concept. We're testing whether retrieval alone solves the core problem before building automation.
QWill this work with TikTok and Instagram?
Long-term yes. Current focus is validating retrieval value. Cross-platform automation is on the roadmap once the core use case is proven.
QIs this an AI cooking app?
No. MyPantry is a retrieval system, not a generator. It doesn't write recipes — it finds the ones you already saved.
QWhy not just use saved folders?
Folders store. MyPantry retrieves. The difference is whether you can find something without knowing exactly where you put it.
